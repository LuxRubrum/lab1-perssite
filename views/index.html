Главная страница

Это - главная страница первой лабораторной работы по Web-программированию. Основной задачей данной работы было спроектировать персональный сайт с элементами списков, ссылок, таблиц и изображений. Также все таблицы стилей вынесены в отдельный CSS-файл.

Структура сайта

На этом сайте Вы найдёте всего четыре страницы: Главную - на которой Вы сейчас находитесь, О себе, Хобби и Достижения. На каждой из этих страниц есть небольшое количество информации, которое расскажет обо мне, как о личности. Я попытался организовать информацию так, чтобы наиболее ясно проявить возможности кода HTML и CSS без каких-либо плагинов или расширений.

Только HTML

Всё, что Вы видите на этом сайте, было оформлено исключительно с помощью возможностей сырого HTML и CSS. С целью полностью выполнить практическое задание, на разных страницах размещены почти все базовые элементы вёрстки: на каждой странице есть таблицы, картинки и ссылки, кое-где встречаются упорядоченные и неупорядоченные списки, а все стили находятся в отдельном файле.
